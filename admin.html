발행하기

여기서 입력하면 GitHub 저장소의 assets/posts.js에 바로 커밋됩니다. 1~2분 뒤 사이트에 반영돼요.

① 저장소 연결 (최초 1회만)

GitHub → 우측 상단 프로필 → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Repository access는 이 저장소 하나만 선택하고, Permissions에서 Contents: Read and write만 켜면 됩니다.
토큰은 이 브라우저에만 저장되고 다른 곳으로 전송되지 않습니다. 공용 컴퓨터에서는 쓰지 마세요.

Google AI Studio에서 무료로 발급받을 수 있어요 (Paley 에이전트 만드실 때 쓰신 것과 같은 키를 그대로 쓰셔도 됩니다). 이 키가 있어야 아래 "PDF로 자동 채우기"가 동작합니다.
